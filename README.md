# Uninet Consultant Website

This is the official website for Uninet Consultant, providing trusted internet solutions.

## Project Overview

This React application showcases Uninet Consultant's services, plans, and company information.

## Development Setup

To run this project locally:

```bash
npm install
npm run dev
```

The application will be available at `http://localhost:5173`.

## Build

To build the project for production:

```bash
npm run build
```

## Key Features

- Responsive design optimized for all devices
- Modern React with TypeScript
- Tailwind CSS for styling
- Component library using Radix UI
- Contact forms and service information
- Performance optimized with Vite

## Project Structure

```
src/
  components/     # Reusable UI components
  pages/         # Main application pages
  lib/           # Utility functions
  hooks/         # Custom React hooks
  assets/        # Images and static assets
```

## Deployment

Build the project and deploy the `dist` folder to your preferred hosting service.
